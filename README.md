# IVP Tree Catalog

## Project Documentation

### Features
- Comprehensive catalog of IVP tree structures
- Search and filter trees based on different criteria
- User-friendly interface for easy navigation

### Installation Instructions
1. Clone the repository:
   ```
   git clone https://github.com/gabriellhtml/IVP-tree-catalog.git
   ```
2. Navigate into the project directory:
   ```
   cd IVP-tree-catalog
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```

### Project Structure
- `src/` - Contains the source code of the application.
- `public/` - Static files such as HTML, CSS, and images.
- `README.md` - Project documentation.

### Formula Explanation
- Details on the formulas used in the IVP trees can be included here, providing mathematical insights and applications.

### Customization Guide
- Guidelines on how to customize the application:
  - Modifying styles in `public/styles.css`
  - Adding new tree structures in the `src/trees` directory.
